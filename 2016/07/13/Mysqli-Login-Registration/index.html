dbconnect.Php

This is a typical connection string to mysql. We will use the information from above to authenticate with the db when we establish a connection. We place it in a seperate file so we can include this wherever we need to connect with the database.

register.Php

This form was created via bootstrap with client side validation but we will not discuss this here. We will also use password_hash($upass, PASSWORD_DEFAULT); as opposed to Md5. This will make the password hash actually secure, since md5 is not.

Index.Php

This is the login page which will ask users to enter email and password to go the home page (members page). The database is connected to via the “dbconnect.php” file. We also use password_verify($upass, $row['password']) to verify that the password instead of “re-hashing” the password using password_hash.

home.Php

If the user successfully logged in, he will be redirected to this “home.php” page. This can be viewed as a members only page only accessible by registered users.

logout.Php

A simple page to logout the users which then redirects them to the login/index page. It destroys the current session.
