Bootstrap Part : Signup Form

Adding input controls with glyphicon for better look. Input control should be within form-group div. To add a glyphicon into an input box, you make another div with class="input-group" then just below the input group make another div with class="input-group-addon". Then add the suitable glyphicon into the addon group div like this <span class="glyphicon glyphicon-user"></span>
